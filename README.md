# Udemy-Clone-PV
# 🎓 Udemy Clone Website

## 📌 Project Overview

This project is a **Udemy Clone Website** built using **HTML and CSS**.
The goal of this project is to practice **frontend web development skills** by recreating the layout of an online learning platform similar to Udemy.

The webpage includes a **navigation bar, course sections, and a clean modern layout** to simulate a real e-learning website interface.

---

## 🚀 Features

* Modern UI design
* Navigation bar similar to Udemy
* Course cards with images and details
* Clean and structured HTML layout
* Styled using CSS
* Beginner-friendly frontend project

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**
